Backend Base URL: http://localhost:8080

PLATFORM OVERVIEW:
LinkedU is an educational networking platform connecting students, teachers, agents, and administrators.
Mission: Democratize access to quality education through direct expert connections.

USER TYPES:
- Students: Learn from professionals in various fields
- Language Teachers: Teach English, French, Arabic, Spanish
- Agents: Recruitment and educational advisors
- Administrators: Manage platform content and users

MAIN API ENDPOINTS:
- POST /api/auth/login - User login
- POST /api/auth/register/guest - Guest registration
- POST /api/auth/register/contract - Professional registration
- GET /api/auth/verify - Email verification
- POST /api/courses - Create new course
- GET /api/destinations - Get travel/study programs
- POST /api/tickets - Support tickets

KEY FEATURES:
- Live Chat with professionals
- Quizzes and assessments
- Study destinations (USA, Canada, UK, France)
- Blog articles on education trends
- Product marketplace with discount codes
- Real-time notifications
- User profiles with credentials verification
